scale

The game engine server is designed with certain scale requirements. The following environments are all designed to run in series, concurrently.

demo

  • 3 concurrent players (1 environments @ 3 users each)
  • 30 registered users

alpha

  • 30 concurrent players (3 environments @ 10 users each)
  • 300 registered users

beta

  • 300 concurrent players (15 environments @ 20 users each)
  • 3,000 registered users

production

  • 3,000 concurrent players (150 environments @ 20 users each)
  • 30,000 registered users
